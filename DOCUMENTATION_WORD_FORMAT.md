## 🔄 Keeping Documents Updated

### If Markdown Files Change
Regenerate Word documents using:
```powershell
pandoc HANDOVER.md -o "WPC_Telangana_Handover_Guide.docx" --toc --toc-depth=3
pandoc SETUP.md -o "WPC_Telangana_Setup_Guide.docx" --toc --toc-depth=3
pandoc ARCHITECTURE.md -o "WPC_Telangana_Technical_Architecture.docx" --toc --toc-depth=3
pandoc DIAGRAMS.md -o "WPC_Telangana_Architecture_Diagrams.docx" --toc --toc-depth=3
```
